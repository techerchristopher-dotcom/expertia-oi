Étape 1 sur 3

Audit digital · État des lieux complet

Comprendre exactement où on en est avant de construire quoi que ce soit.

Problème — la structure a 1 an d'existence, une page Facebook active, mais aucun outil digital pour gérer les commandes. Les pros passent commande par capture d'écran WhatsApp envoyée aux gérants — pas de traçabilité, pas d'historique, pas de catalogue structuré.

Méthode — audit complet en 3 volets : analyse de l'existant digital (Facebook, présence web, outils internes), cartographie des flux de commande actuels, et analyse de la concurrence locale. L'objectif est d'identifier les points de friction et de poser les bases du cahier des charges.

Livrable — un rapport d'audit structuré qui liste les constats, les priorités et les recommandations — base de travail pour l'étape 2.

Constats identifiés

Zéro outil de commande structuré

Les commandes arrivent par WhatsApp sous forme de captures d'écran. Aucune traçabilité, risque d'erreurs, charge mentale élevée pour les gérants.

Catalogue produit non digitalisé

Pas de liste produits en ligne, pas de prix publics accessibles. Le client ne peut pas anticiper sa commande sans contact direct.

Page Facebook existante mais sous-exploitée

Une présence Facebook est en place — point d'appui pour la stratégie digitale, mais sans contenu régulier ni lien vers un site ou catalogue.

Marché B2B uniquement — ouverture B2C planifiée

Clientèle 100 % professionnelle aujourd'hui. L'ouverture aux particuliers dans 6 mois impose une architecture e-commerce qui gère deux types de profils clients dès le départ.

BLOCAGE IDENTIFIÉ · AUDIT

PROBLÈME

Le logiciel de gestion PEM n'a aucune API

Impossible de connecter directement le logiciel interne au site web. Les stocks, prix et références produits sont dans une base de données inaccessible depuis l'extérieur.

→

SOLUTION RETENUE

Agent IA de synchronisation automatique

Un agent IA tourne en tâche de fond et extrait les données de la base toutes les 10 minutes — stocks, prix, références — pour les réinjecter automatiquement dans le site web. Résultat : catalogue toujours à jour, sans intervention manuelle.
